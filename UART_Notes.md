What is UART?
UART (Universal Asynchronous Receiver Transmitter) is a serial communication protocol used for communication between two devices.
IT is two wire communication Protocol
 - One wire is for Transmission 
 - Second wire is for Reciver
Data formet & Transmission speed are configarable

Transmission and interaction does not happen in real time, We can say that in technically They did not get any signal from the output.
* UART Transmission speed is slower compared to parallel communication but it Need bus interface for Communication

Features

- Asynchronous communication
- Full duplex communication
- Uses TX and RX lines
- No clock signal required

UART Pins

- TX (Transmit)
- RX (Receive)

UART Frame Format

1. Start Bit
2. Data Bits (5-9 bits)
3. Optional Parity Bit
4. Stop Bit

Baud Rate

Baud rate is the speed of data transmission.
Examples:

- 9600
- 19200
- 115200

Applications

- Microcontrollers
- GPS Modules
- Bluetooth Modules
- Serial Communication

Advantages

- Simple to implement
- Low cost
- Widely supported
